PARAMETRAGE

Sans être directement lié à la saisie des dossiers, le menu paramétrage est néanmoins d'une grande utilité. En effet, il vous permet de personnaliser le logiciel installé sur votre machine, en particulier sur les points suivants :

  • Coordonnées et logo de votre organisme ; ils figureront non seulement sur la page de couverture des dossiers que vous imprimerez, mais aussi sur votre écran d'accueil ;
  • Personnalisation des intitulés du compte de résultat, de la trésorerie, du bilan et éventuellement des taux de TVA (5 paramétrages possibles) ; ces fonctionnalités sont destinées à vous permettre de réaliser une présentation qui "colle" au plus près à votre dossier ;
  • Paramétrage de trois niveaux d'O.C.C. Différents générant des intérêts sur découvert en cas de solde de trésorerie négatif (c'est le solde poyen annuel qui est pris en compte dans le calcul) ;
  • Personnalisation des impressions (ordre et titres du sommaire) ;
  • Actualisation des taux de cotisations sociale (5 paramétrages possibles) ;
  • Mise à jour du barème de l'impôt sur le revenu (5 paramétrages possibles).

    Pour accéder aux paramétrage du logiciel aucun dossier ne doit être ouvert. Dans la barre de menu en haut de l'écran, cliquez sur Dossier, puis choisissez l'option Paramétrage. Un certain nombre de boutons permettent d'accéder aux différentes fonctionnalités du module.

    L'écran d'accueil offre deux grandes familles de possibilités :

  • Le paramétrage du logiciel ;
  • La sauvegarde des paramètres que vous avez défini.

    Sauvegarde.

    Une série de boutons située en bas de l'écran permet de lancer des actions de sauvegarde. Suivant les cas, celles-ci peuvent être soit partielles, soit globales.

    Les boutons Sauvegarde des référentiels et Restauration des référentiels ont une action partielle. Les mises à jour qu'ils lancent prennent simplement en compte les référentiels des cultures et des cultures pérennes.

    Les boutons Sauvegarde du paramétrage et Restauration du paramétrage ont une action globale. Les mises à jour qu'ils lancent prennent en compte non seulement les paramètres du logiciel tels que vous les avez définis, mais aussi les différents référentiels.

    Attention

    Si vous disposez déjà de référentiels pour SILEX il est préférable d'installer dans un premier temps les paramétrages et les référentiels après.

    Le bouton Restauration param. CER43 permet de restaurer la configuration d'origine. Il n'agit que sur les paramètres du logiciel et pas sur les référentiels cultures et cultures pérennes.

    Paramétrage du logiciel .

    Il est définit à travers 7 options: Logiciel ; Résultat BA ; Résultat IS ; OCC ; Impression ; MSA ; Impôts.

    Attention

    A chaque écran vous devez impérativement valider chaque modification apportée à l'aide du bouton Enregistrer présent à tous les niveaux.

    Logiciel

    En cliquant sur le bouton on découvre le tableau suivant. Il comprend trois onglets : Organisme, Répertoires, Mot de passe.

    Organisme

    Ce tableau reçoit l'adresse de l'organisme ainsi que le logo. Celui-ci est un point BMP qu'il faut placer dans le répertoire WinSilex/icons. En appuyant sur le bouton de droite on a accès à la boite suivante pour charger le logo.

    Répertoire

    Emplacement des répertoires

    Cette partie de l'onglet reçoit les adresses des répertoires où les données sont stockées par défaut.

    Création d'un nouveau dossier

    Lorsque vous créez un nouveau dossier dans Silex, celui-ci peut être vide de toute donnée. Les cases libres en saisie seront alors toutes vides. En cochant l'option concernée, vous pouvez également choisir la création à partir d'un dossier source. Dans ce cas, lorsque vous créez un nouveau dossier, Winsilex charge une copie d'un "dossier source" qui contient déjà un certain nombre de renseignements. Certaines valeurs apparaissent ainsi saisies par défaut dans le dossier que vous venez de créer. Pour pouvoir utiliser cette possibilité, vous devez posséder un "dossier source" (vous pouvez soit créer ce dossier, soit utiliser ceux fournis avec le logiciel, notamment : CER-43.slx) et indiquer son chemin d'accès (Nom du dossier source et Répertoire).

    Divers

    Vous pouvez indiquer ici les valeurs par défaut de quelques données.

    Le Nb d'années de l'étude par défaut et la Devise par défaut pourront également être modifiées directement dans les dossier.

    Le Calcul automatique de la S.A.U. peut être désactivé si votre matériel n'est pas assez puissant. A chaque modification de l'assolement une relecture de la fiche culture sera nécessaire.

    Résultat BA

    Ce tableau recouvre 2 onglets : Taux de TVA et Configuration résultat - bilan

    Un bandeau en haut du tableau permet de choisir entre cinq paramétrages qui peuvent être personnalisés de façon différente. Chacun d'eux doit être nommé : par exemple, le Paramétrage n° 1 s'appelle EPI-PAM. L'un des cinq paramétrages est défini comme Paramétrage utilisé par défaut en cochant la case correspondante. C'est donc ce paramétrage qui sera retenu lors de la création de tout nouveau dossier. Vous avez toutefois la possibilité de modifier votre choix à l'intérieur du dossier lui même (par exemple, lorsque vous devez traiter des cas particuliers).

    Taux de TVA

    En cliquant sur "le bouton radio" à coté de Produits, Charges opé., Frais de structure etc..., des tableaux correspondant aux différents postes sont donnés. Tous les taux de TVA sont modifiables sur les lignes. Un ascenseur à droite du tableau permet le déplacement.

    En ce qui concerne le poste Immo.-Emprunts, les valeurs des paramètres sont également modifiables directement dans le dossier en utilisant le module Immo.-Emprunts.

    Mode de rbt de la TVA

    Le menu déroulant permet de choisir entre deux options différentes de remboursement de la TVA :

    1- Rbt TVA avec délais (mois) ; par exemple, dans le cas d'un remboursement au mois ou au trimestre ;

    2- Rbt TVA annuel (an. Civ.) ; dans le cas d'un remboursement annuel de la TVA.

    Suivant l'option que vous avez retenue, des paramètres sont ensuite à renseigner :

    • Le Délai de rbt de la TVA (si option 1) ; il est indiqué en mois ;
    • Le Mois de remboursement de la TVA (si option 2) ; dans l'exemple ci-dessus, le crédit TVA est remboursé en Juin.

      N.B. : ces options ne concernent que la TVA relative aux immobilisations. Pour les opérations courantes, le mode de remboursement de la TVA est toujours annuel.

      Vous pouvez également paramétrer le Mois de plantation. Cette valeur est utilisée dans le cas où votre dossier comprend des cultures pérennes. Elle sert de point de départ au calcul du crédit TVA dans le cas où un investissement est réalisé dans une plantation.

      En cochant la case Calcul ADIs sur KI déb., le calcul des assurances décès et invalidités sera fait par défaut sur Le capital début. Toutefois une modification de ce calcul est possible sur chaque emprunt dans le dossier.

    Configuration résultat - bilan

    Dans le logiciel ce tableau est sans aucun doute "une pièce" importante. Il permet une modification de tous les libellés du Compte de Résultat du Tableau de Trésorerie et du Bilan.

    L'accès à ces différents comptes se fait avec le menu déroulant sur la droite.

    L'ascenseur permet de se déplacer dans la liste.

    L'invisibilité : Si la case est cochée la ligne n'apparaît pas à l'écran et à l'impression. Toutefois, Si une valeur est donnée pour cette rubrique l'invisibilité sera rompue. Cette option permet de ne pas faire apparaître dans l'étude de libellé pour lesquels aucune donnée n'a été saisie. Au contraire, il est possible de faire apparaître certains libellés de façon obligatoire, même quand la valeur correspondante est nulle.

    Par exemple : vous pouvez définir un paramètrage spécial pour des exploitations spécialisées en cultures végétales (grandes cultures ou cultures pérennes) de façon à ce qu'aucun libellé concernant des animaux n'apparaisse à l'impression.

    Résultat BIC

    Dans le cas où vous avez fait au départ le choix d'une Société à l'IS (cf. § Généralités), le jeux de cinq paramétrages possibles est différent du cas où vous faites une simulation sur une entreprise soumise à l'IR. Les cas envisagés sont destinés à couvrir non seulement les entreprises agricoles soumises à l'impôt sur les sociétés, mais aussi toutes les activités commerciales qui peuvent relever de ce régime.

    Ce tableau sert donc à paramétrer les résultats pour les cas où le logiciel est utilisé dans le cadre d'une problématique d'impôt sur les sociétés. En dehors de cet aspect, il correspond a une simple duplication du module Résultat BA. Son fonctionnement est donc en tout points identique.

    OCC

    Différents seuils pour le calcul du découvert bancaire sont proposés ainsi qu'un taux d'intérêts correspondant. Les seuils sont appliqués sur le solde moyen de la trésorerie. Si celui-ci est négatif des intérêts sont calculés.

    De la même façon que pour les tableaux Résultat BA et Résultat BIC, vous pouvez personnaliser différemment les trois paramétrages qui vous sont proposés. Le bandeau en haut du tableau est le même que dans ces deux tableaux.

    MSA

    De la même façon que pour les tableaux Résultat BA et Résultat BIC, vous pouvez personnaliser différemment les cinq paramétrages qui vous sont proposés. Le bandeau en haut du tableau est le même que dans ces deux tableaux.

    Plusieurs tableaux sont accessibles au moyen des boutons situés à droite de l'écran. Ils permettent de renseigner l'ensemble des critères nécessaires au calcul de la simulation des cotisations MSA.

    Le bouton symbolisant une calculette en bas à droite de l'écran permet de lancer directement le module de simulation des cotisations MSA afin de tester les critères que vous venez de saisir.

    Impôt

    Sur le même principe que le tableau MSA, deux onglets vous permettent de saisir l'ensemble des critères techniques nécessaires au calcul de la simulation de l'impôt sur le revenu, et cela pour chacun des cinq paramétrages possibles.

    Impression

    De la même façon que pour les tableaux Résultat BA et Résultat BIC, vous pouvez personnaliser différemment les cinq paramétrages qui vous sont proposés. Le bandeau en haut du tableau est le même que dans ces deux tableaux.

    Configuration impression du dossier

    Il y a 60 lignes à renseigner. L'ordre d'impression est identifié par un numéro dans la colonne correspondante, il peut être modifié en respectant certaines règles :

    • Les modules à imprimer sont regroupés en blocs ; Le libellé des blocs est inscrit en caractères gras alors que les simples modules apparaissent en gris ;
    • Un module peut correspondre à une page imprimée et se suffire à lui même ou bien n'être qu'un titre sous lequel se regroupent plusieurs modules ; pour cette raison, les modules ne doivent pas être séparés du bloc auquel ils appartiennent (par contre, il est tout à fait possible de modifier l'ordre d'impression des modules à l'intérieur d'un bloc).

      Exemple :

      Résultats : Compte résultat - Trésorerie - Bilan

      Le bilan ne peut pas être dans un autre bloc. Mais l'ordre à l'intérieur de ce bloc peut être modifié.

      Pour plus de facilité de travail, il faut d'abord modifier l'ordre des blocs puis l'ordre dans le bloc.

      Après avoir personnalisé l'ordre d'impression, vous pouvez remettre tous les éléments dans l'ordre au moyen du bouton Tri.

      La colonne sous le titre Impr. O/N permet de choisir les éléments que vous voulez pouvoir imprimer et ceux que vous ne voulez jamais voir apparaître sur les documents.

      Vous pouvez créer votre "dossier type" en cochant les cases de la colonne Impr. par défaut. Ce paramétrage est rappelé par le bouton correspondant dans le module d'impression du dossier.

    Impression des marges brutes

    Dans un dossier vous pouvez affecter les marges animales sur la surface fourragère.

    A l'impression si vous souhaitez que ce document soit inséré après chaque activité animale, vous devez cocher la case Imp. MB animal / SF.

    Divers

    Votre logo peut apparaître systématiquement sur la page de garde si la case Imp. du logo sur page de garde est cochée.

    Le chemin du dossier peut apparaître systématiquement sur la page de garde si la case Imp. du chemin dossier sur page de garde est cochée.

    Synthèse du dossier

    Le menu déroulant permet de sélectionner la synthèse que vous souhaitez avoir par défaut lors de la création d'un dossier. Ce paramètre peut également être modifié directement dans le dossier.

    En pratique, la synthèse correspond à une page de critères qui balayent de façon rapide les chiffres clés de l'étude. Vous disposez de plusieurs modèles de synthèse parmi lesquelles vous avez la possibilité de choisir celle qui vous convient le mieux.
