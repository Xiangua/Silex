       Présentation du logiciel Sil-Docs  
    
 
  Objectif :

Fournir aux utilisateurs de Silex un outil générant automatiquement des documents RTF basés sur une étude prévisionnelle.


  Principe de fonctionnement :

Sil-Docs génère le document résultat à partir d'un modèle du document.
Le modèle est un simple document RTF dont le contenu correspond au document à obtenir SANS les données. A l'emplacement des données à faire paraître dans le document créé, le modèle contient un code spécifique entre les balises #slx( et )#.

 

 

  Sil-Docs dans la pratique :

Sil-Docs se présente sous la forme d'une barre d'outils avec des menus déroulant. Pointez la souris sur un menu pour l'ouvrir.

- Création d'un document

Ouvrir le menu "CREATION DOC" de la barre d'outils Sil-Docs et cliquer "GENERER". 

Sélectionnez le modèle de document dans la partie gauche de la fenêtre.

Choisissez à droite le répertoire destination dans lequel sera généré le document RTF. Sil-Docs choisit par défaut le répertoire du dernier dossier Silex sélectionné. Vous pouvez choisir un autre dossier d'étude grâce au bouton "dossier". Les données du document créé seront ainsi basées sur le dossier Silex choisi. 

Cliquez enfin le bouton "Créer doc". Une barre de progression s'affiche alors, et le document créé s'ouvre automatiquement dans l'éditeur de texte que vous avez lié à Sil-Docs.

A noter : la création d'un document peut se faire directement dans win-Silex en cliquant le bouton "Sil-Docs" après ouverture d'un dossier d'étude.
