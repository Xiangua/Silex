       Importer / exporter des modèles  
    
 

Vous pouvez importer / exporter des modèles de graphique ou de tableau en cliquant "AJOUT" / "COPIE" dans le menu "OUTILS". 

Astuce : pour dupliquer un modèle, copiez-le dans un autre dossier grâce à "COPIE" puis ajoutez-le à votre liste de modèles grâce à "AJOUT" depuis le dossier en question. Ne tenez pas compte du message indiquant que le modèle existe déjà dans votre liste ! Plusieurs modèles identiques peuvent co-exister sans problème dans Sil-Docs.

 

 

Cette fenêtre permet également de supprimer un ou plusieurs modèles dans le répertoire de destination ( mode Copie ) ou dans la liste des modèles ( mode Ajout ). Attention ! la suppression est définitive ( les modèles supprimés ne seront pas récupérables dans la "poubelle" ).
