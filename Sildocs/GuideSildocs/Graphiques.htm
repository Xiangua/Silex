       Construction des modèles de graphique  
    
 

  Insérer un graphique dans votre modèle de document :

Vous pouvez insérer dans vos modèles de document des modèles de graphique. Pour cela, cliquez "GRAPHIQUE" dans le menu "INSERTION". Sélectionnez un graphique dans la liste de gauche et cliquez le bouton "INSERER" en bas à droite. 

 

 
  Éditer vos propres modèles de graphique :

Cliquez le bouton en haut à gauche de l'éditeur des graphiques pour créer un nouveau graphique. Définissez votre graphique et cliquez "Enregistrer" dans l'onglet Paramétrage.

Commencez par sélectionner les séries de données à afficher en les double-cliquant dans l'explorateur de gauche de l'onglet Paramétrage. Les séries double-cliquées sont ajoutées dans la liste des séries du graphique. Un modèle de graphique peut comporter jusqu'à dix séries de données.

Pour supprimer une série de données du graphique, cliquez-la dans la liste de droite de l'onglet Paramétrage, puis cliquez le bouton "Retirer cette série du graphique :" .

Ensuite, choisissez le type d'affichage des séries ( en deux ou trois dimensions ) :

  Paramétrage des graphiques :

Sil-Docs propose un paramétrage complet de vos modèles de graphiques ( une cinquantaine de paramètres ). Vous pouvez lui affecter un titre, un commentaire, une légende qui peuvent être déplacés et redimensionnés dans l'onglet Visualisation. Utilisez l'onglet "Autres paramétrages" dans l'onglet général de paramétrage pour personnaliser le texte affiché avec le graphique ( contenu, orientation, taille, retour à la ligne ).

 

  Options de l'onglet Visualisation :

L'éditeur permet de sélectionner deux axes des Y pour votre graphique lorsque ses séries de données n'ont pas les mêmes unités de valeur. Cliquez une série de données dans l'onglet Visualisation pour savoir quel axe des Y elle utilise ( indiqué en vert au-dessus du graphique ). Double-cliquez là pour lui affecter l'autre axe des Y.  

Pour faire une rotation des graphiques en 3D, maintenez enfoncée la touche "Ctrl" et orientez le graphique avec la souris.
