       Paramétrage de Sil-Docs  
    
 
Dans le menu "PROPRIETES" de Sil-Docs, vous avez accès au paramétrage du logiciel.

Vous pouvez lier Sil-Docs à un autre traitement de texte que celui indiqué en cliquant "AUTRE CHOIX". Les éditeurs de texte sont alors recherchés sur votre ordinateur. Si aucun éditeur n'a été trouvé sur votre machine, vous pouvez indiquer manuellement son chemin à Sil-Docs en cliquant le bouton "Chercher". 

A noter : Sil-Docs dans sa version 2003 gère uniquement MS Word ( versions 97 et ultérieures ) et Lotus Word Pro ( versions 9.5 et suivantes ). 

 

 

Dans le menu "PROPRIETES", vous avez également le choix d'insérer le code Sil-Docs dans vos modèles de document à la suite du curseur ou à la ligne suivant le curseur.

"EDITEUR DE TABLEAU" lance la fenêtre de paramétrage de l'éditeur des modèles de tableau.
