       Le modèle de document  
    
 

Sil-Docs propose une liste de modèles types pour la création de vos documents.
Vous pouvez ajouter à cette liste vos propres modèles de document grâce à l'éditeur convivial de Sil-Docs.

Les modèles sont sauvegardés au format texte enrichi RTF mais avec l'extension SLM ( Silex Model ), afin de les différencier des documents résultats.
De ce fait, sauvegardez vos modèles depuis Sil-Docs ( comme expliqué dans le paragraphe suivant ) et non directement avec le traitement de texte.
  Premier pas :

Cliquez le bouton "NOUVEAU" dans le menu "DOCUMENT". Il vous suffira de cliquer "SAUVER (SOUS)" dans le même menu et de choisir le type "Fichier Modèle (*.slm)" pour enregistrer votre document comme modèle. 

Éditez  votre modèle comme bon vous semble avec les outils de votre traitement de texte ( MS Word ou Lotus Word Pro toutes versions ) auquel viennent se joindre les outils du menu "INSERTION" de Sil-Docs :

  Insérer le code désignant une donnée :

Cliquez le bouton "DONNEE" du menu "INSERTION". Le premier onglet permet d'insérer une valeur, une année d'étude ou le libellé d'une série de données. Dans tous les cas, sélectionnez d'abord une série de données Silex. 

 

Le deuxième onglet permet d'insérer un champ Silex en le sélectionnant dans l'une des quatre listes proposées.

Lorsque vous cliquez ensuite le bouton "INSERER", Sil-Docs se charge de générer le code spécifique à votre choix et l'insert dans le modèle en cours à l'emplacement du curseur. 

Ne modifiez pas le texte du code généré afin que Sil-Docs puisse l'interpréter lors de la création d'un document résultat. Vous pouvez toutefois changer sa mise en forme ( couleur, taille, style, etc ) qui sera appliquée au texte remplaçant le code dans le document résultat.  

  Insérer la date ou l'heure :

Vous pouvez également afficher la date et/ou l'heure grâce au bouton "DATE" du menu "INSERTION". La date/heure sera celle de la création du document résultat si vous cochez "Mettre à jour automatiquement".
